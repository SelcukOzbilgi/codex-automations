# Email Digest — Skill Setup

This skill teaches Codex how to scan Gmail, label matching emails, and send a clean daily digest. Install it once — then trigger it anytime via an automation.

---

## Installation

**1. Open Codex and go to Skills**

**2. Create a new skill named `email-digest`**

**3. Paste the skill prompt below into the skill editor**

**4. Save**

That's it. The skill is now available to any automation in your Codex workspace.

---

## Skill Prompt

```
# Skill: email-digest

## Purpose
Scan today's Gmail inbox for emails matching defined keywords, label each match, summarize their contents, and send a clean digest back to the same inbox.

## Inputs
When this skill is called, the following three variables must be provided:

- SOURCE_KEYWORDS — comma-separated keywords to search in email subject or body
- SOURCE_LABEL — Gmail label to apply to every matching email
- DIGEST_LABEL — Gmail label to apply to the digest email

## Tools
Use Composio for all Gmail operations. Required tools:
- GMAIL_FETCH_EMAILS
- GMAIL_ADD_LABEL_TO_EMAIL
- GMAIL_CREATE_EMAIL_LABEL
- GMAIL_SEND_EMAIL

## Steps

### 1. Search Gmail
- Use GMAIL_FETCH_EMAILS
- Search today's emails where subject OR body contains any of the SOURCE_KEYWORDS
- Always use today's date dynamically — never hardcode a date
- Collect per email: sender name, sender address, subject line, full body

### 2. Label matching emails
- For each matching email:
  - If SOURCE_LABEL does not exist, create it with GMAIL_CREATE_EMAIL_LABEL
  - Apply SOURCE_LABEL using GMAIL_ADD_LABEL_TO_EMAIL

### 3. Summarize each email
Extract per email:
- Sender name and address
- Subject line
- 2–4 key points (what the email is actually about)
- Any deadlines, events, or dates mentioned
- Any action items (things that need a response or follow-up)
- Anything forward-looking (upcoming opportunities, announcements, deadlines)

### 4. Compose the digest email
Use this exact format:

---
Subject: 📬 [SOURCE_LABEL] Daily Digest — [TODAY'S DATE]

[Repeat for each email found, in order received:]

**[#]. [Sender Name] <[sender@email.com]> — [Subject Line]**
- [Key point 1]
- [Key point 2]
- [Action item or deadline if any]
- [Forward-looking note if any]

---
⚡ Highlights:
[2–3 sentences summarizing what matters most across all emails today]

🎯 Action required:
[Bulleted list of anything needing a response or action. If nothing: "Nothing requiring action today."]

---
*Generated by email-digest skill.*

---

### 5. Send the digest
- Send TO yourself (same authenticated Gmail account)
- After sending, apply DIGEST_LABEL to the sent digest email
- If DIGEST_LABEL does not exist, create it first with GMAIL_CREATE_EMAIL_LABEL

### 6. Edge cases
- No matching emails found → send "No [SOURCE_LABEL] emails today." and still apply DIGEST_LABEL
- Gmail search fails → stop gracefully and log the error

## Notes
- Always use today's date dynamically
- Never hardcode dates, email addresses, or label names
- Handle Unicode and special characters correctly (accented letters, non-Latin scripts)
- Write the digest in the same language as the source emails
```
